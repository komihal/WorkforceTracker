Политика конфиденциальности

Последнее обновление: 18 сентября 2024 г.

1. Введение

WorkforceTracker - это мобильное приложение для отслеживания рабочего времени сотрудников. Данная политика конфиденциальности описывает, как мы собираем, используем и защищаем вашу личную информацию.

2. Собираемая информация

2.1 Геолокационные данные

Приложение запрашивает следующие разрешения для работы:

  • ACCESS_FINE_LOCATION - для точного определения местоположения
  • ACCESS_COARSE_LOCATION - для приблизительного определения местоположения
  • ACCESS_BACKGROUND_LOCATION - для отслеживания местоположения в фоновом режиме

Геолокационные данные используются исключительно для:

  • Отслеживания времени прихода и ухода с работы
  • Подтверждения нахождения сотрудника на рабочем месте
  • Автоматической регистрации рабочего времени

2.2 Камера

Приложение использует разрешение CAMERA для:

  • Создания селфи-фотографий при входе/выходе с работы
  • Дополнительной верификации личности сотрудника

2.3 Уведомления

Разрешение POST_NOTIFICATIONS используется для:

  • Уведомлений о начале/окончании рабочего дня
  • Напоминаний о необходимости отметки
  • Важных уведомлений от системы

2.4 Другие разрешения

  • INTERNET - для синхронизации данных с сервером
  • READ_MEDIA_IMAGES - для доступа к сохраненным фотографиям
  • ACTIVITY_RECOGNITION - для определения движения пользователя
  • WAKE_LOCK - для работы в фоновом режиме

3. Использование данных

Собранные данные используются для:

  • Учета рабочего времени сотрудников
  • Генерации отчетов для работодателя
  • Обеспечения безопасности и контроля доступа
  • Улучшения функциональности приложения

4. Передача данных

Данные передаются только:

  • На сервер работодателя для обработки и хранения
  • В зашифрованном виде через защищенное соединение
  • Только с вашего явного согласия

5. Безопасность данных

Мы принимаем все необходимые меры для защиты ваших данных:

  • Шифрование данных при передаче
  • Безопасное хранение на защищенных серверах
  • Ограниченный доступ к данным
  • Регулярное обновление систем безопасности

6. Хранение данных

Данные хранятся в течение времени, необходимого для:

  • Выполнения трудовых обязательств
  • Соблюдения требований законодательства
  • Генерации отчетов и аналитики

7. Ваши права

Вы имеете право:

  • Получить информацию о собранных данных
  • Исправить неточные данные
  • Удалить ваши данные (в соответствии с политикой компании)
  • Отозвать согласие на обработку данных

8. Изменения в политике

Мы можем обновлять данную политику конфиденциальности. О существенных изменениях мы уведомим через приложение или по электронной почте.
